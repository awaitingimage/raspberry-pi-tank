# raspberry-pi-tank

This project is the server and client app required to drive a tank via a browser.
![Tank](./static/images/tank.jpg)

## Prerequistes

- Raspberry Pi
- Node install on the Pi
- Connected Dual H Bridge
- Tank & motors

![Tank guts](./static/images/tank-guts.jpg)

## Getting started

```javascript
 npm run host-mjs
```
